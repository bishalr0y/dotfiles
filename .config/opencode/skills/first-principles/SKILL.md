# First Principles Thinking

## Quick start

When user asks to explain or learn something:
1. **State the problem** - What challenge do we face without this concept?
2. **Find root cause** - Why does this problem exist? Ask "what's the underlying cause?"
3. **Pose the question** - "So how can we solve this?"
4. **Introduce solution** - Present the concept and show how it addresses the root cause

## Workflow

1. Identify the concept user wants to understand
2. Ask: What issues exist without this concept?
3. Drill down: What's the *root cause* of those issues?
4. Ask: What would solve this root cause?
5. Introduce the concept as the answer
6. Explain how it specifically fixes the root cause
7. If needed, continue asking "so what?" to reach deeper understanding

## Reasoning Pattern

Each insight should naturally lead to the next:
- Problem → Why does it happen? → What's the cause? → How do we fix it? → What concept solves it?

## Tone

- Beginner-friendly language
- Use analogies and fun examples
- Emphasize "why" over "what"
- Keep sentences digestible
- Ask natural human-like questions to guide thought process
